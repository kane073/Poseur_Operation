Addition.js

Ce fichier contient le code correspondant aux calculs liés au module d'addition du programme.

Les variables

Variable Description
listArgumentVariable contenant les arguments passés lors de la création de l'addition
operandeTableau qui contient l'ensemble des opérandes de l'addition
retenuesTableau contenant les retenues
resultatNombre contenant le resultat de l'addition

Les fonctions

Fonction Description
remplissageOperande()Remplit le tableau operande avec le contenu de listArguments
verifieNumbreArgument()Vérifie que le nombre d'opérandes est compris entre 2 et 10
verifieLongueurDeChaqueArgument()Vérifie que chaque opérande contient entre 1 et 18 chiffres
longueurMaxPartieEntiere()Calcul la longueur de la partie entière la plus grande parmis les opérandes
longueurMaxPartieDecimale()Calcul la longueur de la partie décimale la plus grande parmis les opérandes
